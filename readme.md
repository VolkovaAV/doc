Install



Как пользоваться:
Первый запуск:
1. Запустить main.exe
2. Нажать кнопку: создать шаблоны. Появится файл: participant_list.xlsx - рабочий файл откуда программа будет подгружать информацию об участниках. Появится папка templates. Внутри нее - 2 файла: bill.doc, act.doc, email.txt
3. Открыть файлы (bill.doc, act.doc) и изменить по своему усмотрению места, выделенные желтым цветом.
!!! Обязательные ключи внутри файлов: !!!
{LAST_NAME}
{MIDDLE_NAME}
{FIRST_NAME}
{SEX}
{SUMM}
{SUMM_NAME}
{F_NAME}
{M_NAME}
{LAST_NAME}
{FILENAME}
список будет пополняться...

Последующие запуски:
1. Добавить необходимых участников в таблицу participant_list.xlsx. 
2. Запустить main.exe
5. Нажать кнопку: сгененировать документы
6. Дождаться надписи: "Генерация документов прошла успешно"
7. Нажать кнопку: отправить документы
8. Дождаться надписи: "Документы отправлены"
